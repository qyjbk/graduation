# 毕设项目

基于Threejs的DIY电脑3D动画网页

## 项目介绍

基于Threejs的DIY电脑3D动画网页，主要使用Threejs和WebGL技术实现，通过使用Threejs的WebGLRenderer渲染器，实现对电脑的渲染，通过使用Threejs的Camera相机，实现对电脑的观察，通过blender建模，将各个电脑模型搭建在Threejs场景中。

## 技术选型
### 框架
- Vue3
- Vite

### 渲染
- Threejs
- gsap

### 后台

## 运行
### 运行环境
- nodejs v16.14.2

### 运行步骤
- 安装nodejs
- 安装依赖
  ```
    npm install
  ```
- 启动项目
  ```
    npm run dev
  ```
